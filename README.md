# 🚀 MERN Stack Internship Project

## 📌 Overview
This repository contains the projects completed during my MERN Stack Web Development Internship. The internship focused on building real-time applications and improving full-stack development skills.

## 📦 Projects Included

### 1️⃣ Real-Time Chat Application
- Built using React, Node.js, Socket.IO, and MongoDB  
- Enables real-time messaging between users  

---

### 2️⃣ Weather Application
- Built using React and OpenWeatherMap API  
- Displays real-time weather data  

---

### 3️⃣ Real-Time Collaboration Tool
- Uses WebSocket and Socket.IO  
- Allows multiple users to collaborate in real-time  

---

### 4️⃣ Productivity Tracker
- Built using React and Chrome APIs  
- Tracks user productivity and usage  

---

## 🛠️ Tech Stack
- MongoDB  
- Express.js  
- React.js  
- Node.js  
- Socket.IO  
- REST APIs
